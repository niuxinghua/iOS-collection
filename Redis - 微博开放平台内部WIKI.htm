安装与Quick Start

yum安装(centos)

  • 在/etc/yum.conf文件尾部加上： 
[sinadsp]
name=sinadsp
baseurl=ftp://dspyum.cluster.sina.com.cn/yum/sinadsp/centos$releasever/$basearch/
enabled=1
gpgcheck=0
skip_if_unavailable=1
metadata_expire=5
  • 然后执行yum install redis 默认会安装当前最新的版本，如需指定版本yum install redis-2.4.20-4

快速启动

  • redis默认安装目录为:/usr/local/sinasrv2/bin/，进入该目录，打开redis.conf，将 
libredis ./libredis.so

修改为 

libredis ./libredis.so.2.4.20.4（这里2.4.20.4为需使用的版本号）

否则出现错误：Error creating tmp file libredis.BbwmAe（随机串）: No such file or directory

  • 在默认目录下，执行 
./redis-server ./redis.conf
正常情况下，服务已启动，可以通过如下命令查看进程是否启动
ps aux | grep redis

连接到redis服务

  • telnet方式，如 
telnet localhost 6379
  • redis-cli方式，如 
./redis-cli  -h localhost -p 6379

更多参数命令 ./redis-cli -h
